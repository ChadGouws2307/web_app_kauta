How to save an Excel file as .CSV



To Save Excel as .CSV


  1. In your Excel spreadsheet, click File.
  2. Click Save As.
  3. Click Browse to choose where you want to save your file.
  4. Select "CSV" from the "Save as type" drop-down menu.
  5. Click Save.

Important Points


  • The file you submit must be a .CSV
  • Ensure each stock column has a name
  • The date column must have a name, called "DATE"
  • The dates of each stock's price must align with the other stock's price dates
  • If you get a server error, it is most likely because your file type is incorrect



File Structure



.CSV file Structure


Header            date,aapl,amzn,fb,nee,baa,cat,peg,air

Period 1          2020/01/01,100,150,60,170,301,122,24,83

Period 3          2020/01/03,112,169,66,174,255,115,23,77

. . .                    . . .

Period n          2020/03/02,109,145,64,176,245,130,25,73



.CSV file Structure Explained


   Header          Ticker symbol for each stock you want to correlate

   Period 1         Each ticker's closing price for the 1st period

   Period 2         Each ticker's closing price for the 2nd period

   Period 3         Each ticker's closing price for the 3rd period

   . . .                  . . .

   Period n         Each ticker's closing price for the nth period
