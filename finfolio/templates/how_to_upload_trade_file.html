Why you should use Trade Files


Trade files are the easiest way to get your trade history into our system. Many brokers will give you a .csv file of all the trades you have made. This can then formatted as shown below and uploaded into our system.It is much easier than using our Trade Form which only allows you to input one trade at a time.
We are constantly adding new features and soon you'll be able to view your portfolio as it currently stands. We plan to also add features that allow you to view you portfolio at historical points in time, so that you can analyse what you did right and wrong, and then improve your investing process.

.CSV file Structure Explained


   Header          Column names for file processing (compulsory)

   Trade 1          Data for the 1st trade

   Trade 2          Data for the 2nd trade

   Trade 3          Data for the 3rd trade

   . . .                  . . .

   Trade n          Data for the nth trade

Important Notes on the File that you Submit


  • The correct data must be in each of the correct columns in your file. If dates are in the Ticker column, the process will fail.
  • Make sure that the ticker symbols are correct - in the future, when we add features that link our different tools together, we want to be able to give you the best information and experience possible.
  • Please note the date format - we know this date format works in our system and have not tested others.
